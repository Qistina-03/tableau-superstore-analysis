# Superstore Sales Analysis Dashboard | Tableau

## Project Overview

This project analyzes sales performance, customer behavior, and regional profitability using Tableau.

The objective is to transform raw sales data into interactive dashboards that provide business insights.

---

## Tools Used

- Tableau Public
- Excel
- Data Visualization
- Calculated Fields

---

## Dataset

Sample Superstore Dataset

Dataset contains:
- Sales transactions
- Customer information
- Product details
- Regional data

---

## Dashboard 1: Executive Sales Overview

Features:
- Total Sales KPI
- Total Profit KPI
- Profit Margin
- Monthly Sales Trend
- Sales by Category
- Sales by Segment

---

## Dashboard 2: Customer & Regional Analysis

Features:
- Top 10 Customers
- Sales by Region
- Profit by State
- Customer Segment Analysis
- Sub-Category Performance

---

## Key Insights

- Identified top-performing product categories.
- Analyzed customer segments contributing to revenue.
- Compared sales and profitability across regions.
